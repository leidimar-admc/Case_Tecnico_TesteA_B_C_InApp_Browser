# 📊Méliuz Shopping - Experimentation & Analytics Framework🩷🛍️

Framework operacional orientado por IA para análise, validação e tomada de decisão em experimentos de produto relacionados a tracking, atribuição, UX mobile e navegação In-App Browser. 

## O projeto inclui:

- framework reutilizável de análise experimental orientado por IA
  -Guia de utilização
  -Arquivos de parametro
  -Prompts para analise, validação e governança
  -Notebook de processamento de dados
   
- análise de teste In- App Browser
  -Análise de contexto do teste
  -Analise do teste e recomendação
  -Proposta de melhoria experimental
  -Apresentação executiva de resultados do teste
  
---

# 🛠️ Ferramentas Utilizadas

| Ferramenta | Objetivo |
|---|---|
| Google Colab | processamento analítico, validações e construção do dataset experimental |
| Python | transformação de dados, agregações, validações e análise estatística |
| Pandas | manipulação e modelagem analítica dos datasets relacionais |
| Matplotlib | geração de visualizações analíticas e gráficos executivos |
| Statsmodels | validação estatística das diferenças de conversão entre variantes |
| Google AI Studio | interpretação experimental, challenge analítico e geração supervisionada de recomendações |
| Gemini | camada cognitiva para interpretação comportamental, causalidade e tomada de decisão |
| GitHub | versionamento, governança, documentação e reprodutibilidade |
| Markdown | documentação metodológica e estruturação do framework |
| CSV | armazenamento dos outputs executivos e datasets analíticos |
